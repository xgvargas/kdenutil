Stabilize
With this function you can convert the points tracked with kdenlive functions Motion-tracker or Auto-mask and generate the invert movement this way you can use the point to stabilize the tracked point instead of following it. Yeah... I need to explain this another way...
Better than that, check out this Ferrolho's blog entry and notice how this function was 99.7% inspirited by his work. While in there also notice how less lazy he is at explaining how to use the function. Amazing!!
